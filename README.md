# Bedprakash Portfolio

Modern React Portfolio Website built using React JS, Vite and Tailwind CSS.

## Tech Stack

- React JS
- Vite
- Tailwind CSS
- React Router DOM
- JavaScript
- HTML5
- CSS3

---

## Features

- Responsive Design
- Modern UI
- About Section
- Projects Showcase
- Contact Section
- GitHub Pages Deployment
- Mobile Friendly Navigation

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Bed-prakash/Portfolio.git
```

Move into project folder:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

---

## Build Project

```bash
npm run build
```

---

## Deploy on GitHub Pages

```bash
npm run deploy
```
